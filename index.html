Dota 2 setup

ClutchReframe Clips needs Dota 2 Game State Integration so it can reliably detect match state, kills, deaths, and assists.

Steam launch option

Add this launch option in Steam, then restart Dota 2.

-gamestateintegration

No clips or empty Library

If the Library is empty after playing Dota 2, start with these checks before contacting support.

1. Start Dota 2

Launch Dota 2 normally after ClutchReframe Clips is installed through Overwolf.

2. Confirm setup

Make sure -gamestateintegration is in Steam launch options, then restart Dota 2.

3. Trigger a highlight

Clips are created after kill, death, or assist moments. Return to the Library after the match or demo session ends.

Recorder conflict

Other Overwolf recording apps may hold the recorder and prevent new clips from being captured.

Recovery steps

  1. Close or disable other Overwolf recording apps.
  2. Fully exit Overwolf from the system tray.
  3. Reopen Overwolf and ClutchReframe Clips.
  4. Restart Dota 2.

Contact support

If the issue continues, email support with a short report so we can identify where the capture flow failed.

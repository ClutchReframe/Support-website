# Support Website 产品描述

## 完整版英文

ClutchReframe Clips Support is the public support website for ClutchReframe Clips, a Dota 2 Overwolf app for capturing gameplay highlights and creating vertical clips.

This website is a technical support entry point, not a separate product website. It provides basic setup and troubleshooting guidance for the current Dota 2-only Overwolf app submission.

Support topics covered:

1. Dota 2 setup — explains that users should add `-gamestateintegration` to Steam launch options and restart Dota 2 so the app can reliably detect match state, kills, deaths, and assists.
2. No clips / empty Library — lists the minimum checks users should perform before contacting support, including launching Dota 2, confirming setup, triggering a kill/death/assist highlight, and returning to the Library after the match or demo session ends.
3. Recorder conflict — explains that other Overwolf recording apps may hold the recorder and prevent new clips from being captured, with recovery steps such as disabling other recording apps, fully exiting Overwolf from the system tray, reopening ClutchReframe Clips, and restarting Dota 2.
4. Contact support — provides the support email `clutchreframe@gmail.com` and asks users to include app version, Overwolf version, screenshots, a short issue description, and reproduction steps.

The support website does not provide user accounts, ticketing, analytics, tracking, advertising placements, or cloud upload features. It does not include Riot verification files and does not make League of Legends support claims. It is intentionally separate from `clips.clutchreframe.com`, which is used for Riot / League of Legends related review material.

ClutchReframe Clips processes gameplay videos and local clip metadata primarily on the user's PC. The app does not automatically upload gameplay videos or local clip metadata to developer servers. The current Overwolf app uses Overwolf platform capabilities and the Overwolf Ads SDK; the support website itself does not display ads.

ClutchReframe Clips is an independent companion app and is not affiliated with or endorsed by any game publisher. Game titles and trademarks belong to their respective owners.
